Access props/theme variables from your styles. Components in this library let you pass a function to any `ss` prop. That enables you to call `(props) =>` only once, then reference props/theme multiple times. All `ss` props accept this function format. Alternatively, use the standard `emotion/styled-components` way to wrap a string in `css`&grave;&grave; or `styled`&grave;` and pass a function to every template literal.
