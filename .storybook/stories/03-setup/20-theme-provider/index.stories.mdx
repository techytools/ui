<Block as="h2" textgradient="accent">1. Install</Block>

This is **not currently published as a standalone library**, but as part of a **mono-repo**. See ["Mono repo" in the stories panel](/docs/get-started-mono-repo--page).   

This UI library `@ps/ui` by design only requires `@emotion/styled` as a peer dependency. NextJS is optional.  
It also uses a couple modules `@ps/fn` and `@ps/contants` from the local mono-repo, but these be removed in a quick refactor.


<Block as="h2" textgradient="accent" ss="margin-top:3rem;">2. Theme</Block>

To get started instantly, with no configuration, import `@ps/ui/components/ThemeProvider` and wrap your app with it.

Or to customize theme and settings, make a copy of `ThemeProvider.tsx`, `theme.ts` and `colors.css` files in your app directory and use that. Here are the files used to customize your colors, sizes, and variables:  

<Code title="components/ThemeProvider.tsx" code={`import { ThemeProvider as EmotionThemeProvider } from '@emotion/react';
  
import theme from '@ps/ui/styles/theme'; // your custom JS variables
  
import 'styles/colors.css'; // your custom CSS variables
  
const ThemeProvider = ({ children }) => {
  return (
    <EmotionThemeProvider theme={theme}>
      {children}
    </EmotionThemeProvider>
  );
};
  
export default ThemeProvider;`} collapsed={true} ss="margin-bottom:0;" />
